# Roso OS

Roso OS is a browser-based desktop simulation inspired by Windows XP. It is not a real operating system; it is a portfolio environment built with React, Vite and Zustand.

## Current stage

The project now includes the Stage 1 technical foundation, Stage 2 window system, Stage 3 XP experience polish, Stage 4 simple apps and Stage 5 filesystem:

- React + Vite JavaScript app
- Zustand UI store
- layered global CSS
- desktop shell
- taskbar
- start button and start menu
- clock
- desktop icons
- open, focus, move, minimize, maximize, restore and close windows
- taskbar entries for open windows
- a demo Mi PC app
- quick launch shortcuts
- improved start menu with all programs
- basic wallpapers and retro UI sounds
- a demo Panel de control app
- Bloc de notas with in-memory editing
- Terminal fake with local commands
- expanded Panel de control sections
- IndexedDB-backed virtual filesystem
- Explorador with folders, text files, rename and delete
- Terminal filesystem commands

Games and mobile support are planned for later stages.
